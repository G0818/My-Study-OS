My Study OS

앱 설치 방법

My Study OS는 휴대폰 홈 화면에 추가하면 일반 앱처럼 빠르게 실행할 수 있습니다. 아래에서 사용하는 기기에 맞는 방법을 따라 해 주세요.

빠른 안내

Summary
안드로이드
Chrome에서 접속한 뒤 오른쪽 위 ⋮ 메뉴에서 앱 설치 또는 홈 화면에 추가를 누릅니다.
아이폰 / 아이패드
Safari에서 접속한 뒤 공유 버튼을 누르고 홈 화면에 추가를 선택합니다.
Android

안드로이드에서 설치하기

Chrome
  1. 안드로이드 휴대폰에서 Chrome을 엽니다.
  2. My Study OS 사이트에 접속합니다.
  3. 오른쪽 위의 ⋮ 메뉴를 누릅니다.
  4. 앱 설치, 홈 화면에 추가, 또는 바로가기 추가를 선택합니다.
  5. 이름이 My Study OS로 되어 있는지 확인합니다.
  6. 설치 또는 추가를 누릅니다.
  7. 홈 화면에 생긴 My Study OS 아이콘으로 실행합니다.
기기나 Chrome 버전에 따라 메뉴 이름이 조금 다를 수 있어요. “앱 설치”가 보이면 그걸 선택하고, 없으면 “홈 화면에 추가”를 선택하면 됩니다.
iOS

아이폰 / 아이패드에서 설치하기

Safari
  1. 아이폰 또는 아이패드에서 Safari를 엽니다.
  2. My Study OS 사이트에 접속합니다.
  3. 화면 아래쪽 또는 위쪽의 공유 버튼을 누릅니다.
  4. 목록에서 홈 화면에 추가를 선택합니다.
  5. 이름이 My Study OS로 되어 있는지 확인합니다.
  6. 오른쪽 위의 추가를 누릅니다.
  7. 홈 화면에 생긴 My Study OS 아이콘으로 실행합니다.
iOS에서는 Chrome보다 Safari에서 홈 화면에 추가하는 것이 가장 안정적입니다.
알림

알림을 사용하려면

Notification
  1. My Study OS에서 Google 계정으로 로그인합니다.
  2. 설정 페이지에서 알림 권한 요청을 누릅니다.
  3. 브라우저 또는 기기 설정에서 알림을 허용합니다.
  4. D-Day 알림 시간과 공부 시작 알림 시간을 설정합니다.
주의: 푸시 알림은 네트워크 연결이 필요합니다. 또한 안드로이드 절전 모드나 iOS의 백그라운드 제한 때문에 알림이 늦거나 오지 않을 수 있습니다.
문제 해결

설치 버튼이 안 보일 때

Help
  1. 브라우저를 새로고침한 뒤 다시 확인합니다.
  2. 안드로이드는 Chrome에서 접속했는지 확인합니다.
  3. iOS는 Safari에서 접속했는지 확인합니다.
  4. 이미 설치되어 있으면 설치 버튼이 다시 보이지 않을 수 있습니다.
  5. 그래도 안 보이면 브라우저 메뉴에서 홈 화면에 추가를 직접 찾아 선택합니다.
설치하지 않아도 웹사이트로 사용할 수 있지만, 홈 화면에 추가하면 앱처럼 더 빠르게 열 수 있습니다.
